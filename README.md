"# AndroidApp" 

정재일 교수님 연구실과 함께한 현대건설 중장비협착방지 관련 앱 개발 프로젝트입니다.
프로젝트 목적: 건설 현장 내 중장비(Anchor)와 작업자(Tag)가 혼재한 상황에서 안전성 확보를 위한 안드로이드 어플리케이션 개발
